# Juan Salazar — Portfolio

Modern, responsive portfolio website built with React + Tailwind CSS.

## 🚀 Live

[juanjosalco.github.io/Portfolio](https://juanjosalco.github.io/Portfolio/)

## Tech Stack

- **React** (Vite)
- **Tailwind CSS v4**
- **Lucide React** (icons)
- **GitHub Pages** (deployment)

## Development

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```
